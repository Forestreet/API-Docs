---
title: "Company Locations"
api: "POST https://api-dev.forestreet.com/markets/{marketId}/companies/locations"
description: "This endpoint retrieves all the locations for companies for a given market."
---

### Path

<ParamField path="marketId" type="string" initialValue="2">
  The ID of the market whose companies you want to retrieve. This can be retrieved by searching for markets.
</ParamField>

### Body Request Object Example

```JSON
{
  "filters": {
    "country": ["United Kingdom"],
    "state": ["Greater London"],
    "city": ["London"],
    "postcode": ["SW4 9EG"],
    "domain": ["jacksonsestateagents.com", "barnardmarcus.co.uk"],
    "dunsNumber": [9962094557]
  }
}
```

### Body

<ParamField body="country" type="array" default="">
  An array of country name strings.
</ParamField>

<ParamField body="state" type="array" default="">
  An array of state strings
</ParamField>

<ParamField body="city" type="array" default="">
  An array of city strings
</ParamField>

<ParamField body="postcode" type="array" default="">
  An array of postcode strings
</ParamField>

<ParamField body="domain" type="array" default="">
  An array of company domains
</ParamField>

<ParamField body="dunsNumber" type="array" default="">
  An array of company DUNS numbers
</ParamField>
